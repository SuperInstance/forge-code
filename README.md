# forge-code

> Source code → tiles → Plato agents.

Decomposes code into tiles: functions, structs, classes, methods, imports, comments. Works on Rust, Python, TypeScript, Go, C, Java. Regex-based, zero AST dependencies, fast enough for agents.

Each tile knows its language, kind, name, and line span. Plato agents navigate codebases room by room.
